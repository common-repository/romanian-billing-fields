== Description ==

Acest plugin adaugă câmpuri de facturare în limba română la WooCommerce Checkout.

Adaugă câmpuri personalizate pentru legislația română:
PF: CNP

PJ:
CIF/CUI
Număr de înregistrare
numarul contului bancar
Numele băncii

Câmpurile sunt afișate în adresa de facturare sub Numele companiei - pe pagina checkout, pagina de mulțumire, comenzile de administrator și e-mailuri.

Puteți modifica aceste câmpuri ca administrator prin comenzi sau ca utilizator din pagina contului meu -> adresele mele.


Vedeți demo aici: <a href="https://tastewp.com/new?pre-installed-plugin-slug=woocommerce%2Cromanian-billing-fields&redirect=plugins.php&ni=true">Romanian Billing Fields Demo</a>

== Installation ==

Consultați [Instalarea pluginurilor](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins) în Codexul WordPress.

Dupa instalare si activare modul nu mai trebuie sa mai faci absolut nimic, campurile o sa apara direct pe pagina de checkout.
